## 第 3 区：资金失衡层来源

用途：解释 `强滞 / 压 / 天 / 滞` 为什么触发，以及是否被更高层压制。

必须字段：

- [已确认] `强滞原始`、`强滞最终`、`强滞代表中位`、`强滞代表K距今`。
- [已确认] `放量下跌原始`、`放量下跌最终`、`天量天价原始`、`天量天价最终`、`放量滞涨原始`、`放量滞涨最终`。
- [已确认] `明显放额`、`连续放额`、`滞涨未推进`、`近3日有效新高`、`创60日新高`。
- [已确认] `收盘位置`、`上影占比`、`涨跌幅`、`K线中位`。
- [记录观察] `资金层内部优先级`：强滞 > 压 > 天 > 滞。

中文读法：

- `强滞` 重点看连续放额后是否仍推不动。
- `压` 重点看高位放额下跌时，收盘是否靠低。
- `天` 重点看是否是真正 60 日新高附近的极端成交额。
- `滞` 是较轻的资金异常提示，若同日有 `强滞 / 压 / 天`，应被压制。

## 第 4 区：动能衰减层来源

用途：解释 `1 / 2 / 3 / 离` 的出现原因，尤其核对成交额参照和动态 `REF`。

必须字段：

- [已确认] `背离等级`、`背离显示`、`背离基础`、`背离高位基础`。
- [已确认] `AMT0`、`均额20`、`AMT0/均额20`、`前三日均额`、`前高距`、`前高成交额`。
- [已确认] `相对均额缩`、`较近3日缩额`、`较前高缩额`、`成交额萎缩`、`成交额枯竭`、`持续缩额`。
- [已确认] `严格MACD顶背离`、`MACD顶背离显示`、`当前价峰`、`前峰距`、`前峰价`、`前峰DIF`。
- [只进验证副图] `动态REF兼容差异_量价`、`动态REF兼容差异_MACD`。

中文读法：

- 量价背离必须同时看基础缩额和前高缩额，不能只看单一缩额条件。
- `前高成交额` 是核心参照；若平台不支持动态 `REF`，再对比兼容近似版。
- `离` 当前是严格峰值版 MACD 顶背离，不能和宽松动能弱化混读。

用途：解释 `弱 / 粘 / 急破 / 破 / 形警` 是否只是轻微震荡，还是结构确实走坏。

必须字段：

- [已确认] `转粘合原始`、`转粘合最终`、`趋势破坏最终`、`急破最终`。
- [已确认] `MA60多头`、`曾经多头`、`多头排列`、`均线粘合`。
- [已确认] `形态预警显示`、`形态短线松动`、`射击星警`、`吞没警`、`乌云警`、`黄昏星警`。

- `弱` 不能只看跌破 MA20，还要看 MA5 是否下压、MA20 破位是否有质量。

用途：本地实验环境批量跑时，用来判断信号有效、偏早、偏敏感还是漏报。

- [已确认] 信号后 3 / 5 / 10 / 20 日最大回撤。
- [已确认] 信号后 3 / 5 / 10 / 20 日是否跌破 MA20、MA60。
- [已确认] 信号后 3 / 5 / 10 / 20 日是否跌破信号 K 中位。
- [已确认] 信号后 3 / 5 / 10 / 20 日是否继续有效新高。
- [已确认] 是否进入更高层级风险信号。

中文读法：

- 单独跌破 MA20 不再作为有效确认，必须结合回撤幅度、跌破信号 K 中位或更高层级风险。
- 如果信号后快速新高且无确认破坏，归入修复类，不直接按误报处理。
