## Read Rules

风险保护视图先看：

```text
黄源 / 黄效 / 市弱 / 前重量 / 前重位 / 今量 / 今位 / 重今% / 黄今% / 热追
```

当前读法：

- `热追=True` 优先视为强风险提示，但要结合 `黄今%` 防止误杀低位修复正例。
- `极弱续高` 不单独显示字段，切到 `市场环境` 视图看 `破60数 / 指10`，再回到 `风险保护` 看 `今量 / 黄今% / 重今%`。
- 不再追第三分支，除非后续出现新的底层证据，而不是从小样本组合里硬挖。
