# 回测计划

本文件用于记录后续回测方案、统计口径和回测后优化结论。

## 初步统计方向

- 启动类信号：启、蓝、黄、速、冲。
- 确认类信号：确2、确3、确4。
- 风险类信号：险、败、警、慎、滞。
- 组合类统计：启动后有确 vs 没有确；启动后出现风险 vs 没出现风险。
- 新旧公式差异统计：原有新版无、新版有原无，分别按启动、确认、风险、慎滞分类。
- 显示空窗统计：源头成立但最终无显示、有效信号成立但无显示、风险源成立但无风险/慎滞显示。
- 未显示原因统计：显示许可、险/败压制、冷却、新鲜度、速后运行、平台高热、强趋势运行、保护期过滤、显示优先级压制。

## 初步收益窗口

- 3根K线；
- 5根K线；
- 10根K线；
- 20根K线。

主公式尚未定稿，暂不开始实际回测。

## 验证方式

后续验证不只依赖人工看图，应结合三层方式：

- 人工看图验证：判断单根K线是否符合走势语境。
- Python 批量统计验证：统计最终显示码、原新版差异码、显示空窗码、未显示原因码后的表现。
- 典型案例回放验证：对争议样本逐项回放源头、过滤、保护、显示和后续走势。
